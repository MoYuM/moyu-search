# Changelog

## [v0.2.1] - 2025-10-24

- **Chore**: 移除不必要的 scripting 权限

## [v0.2.0] - 2025-10-24

### Features

- **Bang Mode**: 新增 Bang Mode 支持（含 GitHub Bang、搜索提示及配置），并完善相关国际化
- **性能优化**: 简化最近 Tab 获取逻辑，显著加速初始化
- **图标系统**: 优化图标缓存策略，增强模糊匹配能力及错误处理
- **快捷键**: 新增快捷键录制功能（支持 Blur 自动停止），完善配置校验
- **体验优化**: 增加反馈入口，统一优化 UI 样式

### Fixes

- 修复 Bang Mode 结果选择逻辑
- 解决 Jira 等网站下的按键冒泡冲突问题
- 修复配置校验逻辑

### Refactor

- 重构代码结构，解耦清理逻辑与 Popup 逻辑

## [v0.0.6] - 2025-08-27

### Features

- **国际化**: 引入多语言支持 (i18n)
- **配置**: 新增快捷键配置项

### CI/CD

- 完善 GitHub Action 构建流程（支持 Tag 触发，修复构建问题）

## [v0.0.4] - 2025-08-10

- **Docs**: 更新项目描述及文档

## [v0.0.3] - 2025-08-02

### Features

- **搜索增强**: 优化无结果时的搜索引擎引导，更新搜索图标
- **交互优化**: 改进全局按键监听机制，完善 Ctrl+P 交互体验
- **UI**: 支持 Popup 外观配置

## [v0.0.2] - 2025-07-27

### Features

- **核心功能**: 实现基础搜索（Tab/书签/历史）及跳转，上线设置页面
- **UI/UX**: 支持暗黑模式，优化搜索列表样式与排序算法
- **性能**: 优化搜索响应速度

### Fixes

- 解决 GitHub 等特定网站的按键冲突及输入框冒泡问题
- 修复图标加载及 Tab 显示相关 Bug
